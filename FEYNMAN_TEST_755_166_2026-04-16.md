## 1. Purpose

The Feynman test for a putative morphism is: if the bridge is real, the same code with a different constraint should produce recognizable signals of the same kind — not by re-tuning, but by re-running. The Prima Materia Framework (PMF) earned a power-morphism stamp on Erdős #30 (Sidon) in 2026-04-15 via the transfer matrix construction T(W) and the Koopman-von Neumann (KvN) identification. This round puts that claim under two stress tests drawn from the Tier-1 amenable list in `CLAUDE.md`:

- **#755** — B_h[g] sequences, a strict parametric generalization of Sidon (B_2[1]).
- **#166** — Sum-free sets, a different constraint style (three-site coupling a + b = c).

The test is: same sidon_transfer_matrix.cpp architecture, swap the local exclusion rule, measure what comes out.

## 2. Method

Both computations reuse the C++ scaffold built for #30 (`sidon_transfer_matrix.cpp`). For #755 the exclusion rule is `diff_count[d] ≤ g` replacing the Sidon `diff_used[d] = bool`. For #166 the exclusion rule is the Schur condition `(a, b, c) with a + b = c forbidden` applied within the window.

State space enumerates all locally legal subsets of `{0, 1, …, W−1}`. The transfer matrix is the bipartite transition graph between window configurations that agree on their overlap. λ_max(T(W)) is extracted via power iteration; the second eigenvalue λ_2 and the gap ratio λ_2/λ_1 are tracked as spectral diagnostics. Data collected for W = 3 … 25.

For #166, a second enumeration computes the global partition function Z(N) = total sum-free subsets of `{1, …, N}` for N = 3 … 28, which directly tests the Cameron-Erdős prediction log₂ Z(N) → N/2 + o(N).

|---|---|---|

Linear fit of log₂(count) vs. N:
- All data (N = 3 … 28): slope = 0.6006, intercept = 1.168

Observed maximum density at N = 28: 14/28 = 0.500 exactly (the extremal family "all integers > N/2" already saturates at N = 28 — and every even N in the data gives density exactly 0.5).

**Feynman test verdict for #166: NEGATIVE CONTROL (as expected).** The local transfer matrix cannot resolve the sum-free constraint because the constraint is non-local. Global enumeration reproduces the known Cameron-Erdős (Green 2004 / Sapozhenko 2003) result. This is a scientifically valuable finding: it demarcates the locality boundary of the PMF/KvN bridge.

## 4. Interpretation

The two tests together are a cleaner demonstration of the bridge's shape than a third positive result on a similar problem would have been. A tool that works on everything is suspicious; a tool with a sharply characterized domain of applicability is a discovery.

**PMF/KvN amenability test (refined from this round):** a problem is amenable iff the constraint is:

1. **Local in a window** — violation is detectable by looking at a contiguous slice of bounded width.
2. **Translation-invariant** — the rule depends on configuration, not position.
3. **Sub-exponential state-space growth** — polynomial TT-rank, so DMRG/tensor methods remain feasible.

#30 Sidon and #755 B_h[g] satisfy all three. #166 sum-free fails condition (1) — the constraint reaches arbitrarily far through the equation a + b = c.

This is the same lesson physics already teaches: nearest-neighbor Ising is transfer-matrix solvable; long-range interactions (1/r^α with α small) are not. The KvN bridge transports that boundary from physics to arithmetic combinatorics.
